# DCF Valuation Model — Trane Technologies
A full-scope Discounted Cash Flow (DCF) valuation model built in Excel.
## What's included
- P&L forecast (multi-year)
- Fixed assets rollforward
- Working capital analysis
- Debt schedule
- Equity schedule
- Firm valuation using WACC/CAPM
- Scenario and sensitivity analysis
## Tools used
- Microsoft Excel (Advanced)
